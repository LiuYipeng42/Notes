### is a

Cat is a Animal，凡是能够满足 is a 的表示**继承关系**

### has a

I have(has) a pen，凡是能够满足 has a 的表示**关联关系**，关联关系通常以**属性**的形式存在。

### like a

Cook(厨师) like a FoodMenu，凡是能够满足 like a 的表示**实现关系**，实现关系通常是**类实现接口**。
